# Make the sample work

The task here is to make the sample work easily and in a reproduce-able way. If this can be achieved with the sample, it should be able to be done with the final product.

## Do it

### Get the prerequisites
```
./prerequisits.comvenience pathToExtractedLeapMotionSDK
```

eg
```
./prerequisits.comvenience /tmp/LeapDeveloperKit_2.3.1+31549_linux/LeapSDK
```


### Build it
```
./compile.convenience
```

### Run it
```
./run.convenience
```
